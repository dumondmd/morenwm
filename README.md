# Instalação

## Pacotes

Pacotes do Composer PHP

```
composer install
```

Pacotes do Node.JS

```
npm install && npm run dev
```

## Migration

Bando de dados **MySQL**

```
php artisan migrate
```

Gere os registros do banco de dados para teste

```
php artisan db:seed --class=AlunoSeeder
```
